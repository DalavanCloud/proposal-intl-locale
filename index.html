1.1.1DeconstructLanguageTag (tag)#

The abstract operation DeconstructLanguageTag an argument tag. It returns a Record with a resolved locale and a list of extensions.

  2. Return result.

1.1.2IsOptionValueSupported (option, value)#

The abstract operation IsOptionValueSupported accepts the arguments option and value. It returns if the given value is a supported value for the given option.

  1. If option equals "calendar", then:
    1. If value matches one of the calendars supported by the system, then:
      1. Return true.
    2. Else,
      1. Return false.
  2. If option equals "collation", then:
    1. If value matches "type" given in Unicode Technical Standard 35 for the collation, except that the values "standard" and "search" are not allowed, while the value "default" is allowed.
      1. Return true.
    2. Else,
      1. Return false.
  3. If option equals "currency", then:
    1. Return IsWellFormedCurrencyCode(value).
  4. If option equals "hourCycle", then:
    1. If value matches a Value column of the row in Table 2, then:
      1. Return true.
    2. Else,
      1. Return false.
  5. If option equals "caseFirst", then:
    1. If value matches one of the values specified in ., then:
      1. Return true.
    2. Else,
      1. Return false.
  6. If option equals "numeric", then:
    1. If value is a Boolean, then:
      1. Return true.
    2. Else,
      1. Return false.
  7. If option equals "numberingSystem", then:
    1. If value matches one of the values in the "Numbering System" column of , then:
      1. Return true.
    2. Else,
      1. Return false.
  8. If option equals "timeZone", then:
    1. Return IsValidTimeZoneName(value).
  9. Throw a RangeError exception.

1.2The Intl.Locale Constructor#

The Locale constructor is a standard built-in property of the Intl object. Behaviour common to all service constructor properties of the Intl object is specified in .

1.2.1Intl.Locale(tag[, options])#

When the Intl.Locale function is called with an argument tag and an optional argument options, the following steps are taken:

  1. If IsStructurallyValidLanguageTag(tag) is false, throw a RangeError exception.
  2. Let locale be the this value.
  3. Let canonicalizedTag be CanonicalizeLanguageTag(tag).
  4. Let r be DeconstructLanguageTag(tag).
  5. Set locale.locale to r.[[Locale]].
  6. If options is undefined, then
    1. Let options be ObjectCreate(%ObjectPrototype%).
  7. Else
    1. Let options be ? ToObject(options).
  8. Let optionKeys be ? options.[[OwnPropertyKeys]]().
  9. For each key in optionKeys,
    1. If key matches a Option column of the row in Table 1, then:
      1. Let value be ? Get(options, key).
      2. If the result of IsOptionValueSupported(fieldName, value) is false, throw a RangeError exception.
      3. Perform ? CreateDataPropertyOrThrow(locale, key, value).
  10. If ? HasOwnProperty(r.[[Extensions]], "u"), then
    1. Let unicodeSubtags be UnicodeExtensionSubtags(r.[[Extensions]]['u']).
    2. Let i be 0.
    3. Let len be ? ToLength(? Get(unicodeSubtags, "length")).
    4. Repeat while i < len:
      1. Let key be Get(unicodeSubtags, ToString(i)).
      2. If key matches a Extension key column of the row in Table 1, then:
        1. Increment i by 1.
        2. Let value be ? Get(unicodeSubtags, ToString(i)).
        3. Let name be the value from a column Option in Table 1, which corresponds to the value key from column Extension keys.
        4. If the result of IsOptionValueSupported(name, value) is false, throw a RangeError exception.
        5. If ? HasOwnProperty(locale, name) is false, then
          1. Perform ? CreateDataPropertyOrThrow(locale, key, value).
      3. Increment i by 1.
  11. SetIntegrityLevel(locale, "frozen").

1.4.3Intl.Locale.prototype.toString ()#

  1. Let loc be the this value.
  2. If Type(loc) is not Object, throw a TypeError exception.
  3. Let locale be Get(loc, "locale").
  4. Let unicodeExt be an empty String.
  5. For each row in Table 1, except the header row, do:
    1. Let name be the name given in the Option column of the row.
    2. Let key be the name given in the Extension key column of the row.
    3. If ? HasOwnProperty(loc, name), then:
      1. Let value be Get(loc, name).
      2. Let kvp be a String value produced by concatenating key, "-", and value.
      3. Set unicodeExt to a String value produced by concatenating unicodeExt and kvp.
  6. If unicodeExt is not empty, then:
    1. Let result be a String value produced by concatenating locale, "-u-", and unicodeExt.
    2. Return result.
  7. Else,
    1. Return locale.
