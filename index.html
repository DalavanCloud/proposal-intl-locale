1Locale Objects#

1.1Abstract Operations for Locale Objects#

1.1.1InitializeLocale (tag, options)#

The abstract operation InitializeLocale accepts the arguments locale, and options. It initializes a new Locale object. It performs the following steps:

  1. If locale.[[InitializedIntlObject]] is true, throw a TypeError exception.
  2. Set locale.[[InitializedIntlObject]] to true.
  3. Let r be DeconstructLanguageTag(tag).
  4. Set locale.[[Locale]] to res.[[Locale]].
  5. If options is undefined, then
    1. Let options be ObjectCreate(%ObjectPrototype%).
  6. Else
    1. Let options be ? ToObject(options).
  7. Let optionKeys be ? options.[[OwnPropertyKeys]]().
  8. For each key in optionKeys,
    1. If key matches a Option column of the row in Table 1, then:
      1. Let value be ? Get(options, key).
      2. If the result of IsOptionValueSupported(fieldName, value) is false, throw a RangeError exception.
      3. Let fieldName be the value from a column Name in Table 1, which corresponds to the value key from column Option.
      4. Set locale.[[fieldName]] to value.
  9. If res.[[Extensions]] has key "u", then
    1. Let unicodeSubtags be UnicodeExtensionSubtags(res.[[Extensions]]['u']).
    2. Let i be 0.
    3. Let len be Get(unicodeSubtags, "length").
    4. Repeat while i < len:
      1. Let key be Get(unicodeSubtags, ToString(i)).
      2. If key matches a Extension key column of the row in Table 1, then:
        1. Increment i by 1.
        2. Let value be ? Get(unicodeSubtags, ToString(i)).
        3. Let fieldName be the value from a column Name in Table 1, which corresponds to the value key from column Extension keys.
        4. If locale.[[fieldName]] doesn't exists:
          1. If the result of IsOptionValueSupported(fieldName, value) is false, throw a RangeError exception.
        5. Set locale.[[fieldName]] to value.
      3. Increment i by 1.

1.1.2IsOptionValueSupported (option, value)#

The abstract operation IsOptionValueSupported accepts the arguments option and value. It returns if the given value is a supported value for the given option.

  1. If option equals "Calendar", then:
    1. If value matches one of the calendars supported by the system, then:
      1. Return true.
    2. Else,
      1. Return false.
  2. If option equals "Collation", then:
    1. If value matches "type" given in Unicode Technical Standard 35 for the collation, except that the values "standard" and "search" are not allowed, while the value "default" is allowed.
      1. Return true.
    2. Else,
      1. Return false.
  3. If option equals "Currency", then:
    1. Return IsWellFormedCurrencyCode(value).
  4. If option equals "hourCycle", then:
    1. If value matches a Value column of the row in Table 2, then:
      1. Return true.
    2. Else,
      1. Return false.
  5. If option equals "CaseFirst", then:
    1. If value matches one of the values specified in ., then:
      1. Return true.
    2. Else,
      1. Return false.
  6. If option equals "Numeric", then:
    1. If value is a Boolean, then:
      1. Return true.
    2. Else,
      1. Return false.
  7. If option equals "NumberingSystem", then:
    1. If value matches one of the values in the "Numbering System" column of , then:
      1. Return true.
    2. Else,
      1. Return false.
  8. If option equals "TimeZone", then:
    1. Return IsValidTimeZoneName(value).
  9. Throw a RangeError exception.

1.2The Intl.Locale Constructor#

The Locale constructor is a standard built-in property of the Intl object. Behaviour common to all service constructor properties of the Intl object is specified in .

1.2.1Intl.Locale(tag[, options])#

When the Intl.Locale function is called with an argument tag and an optional argument options, the following steps are taken:

  1. If IsStructurallyValidLanguageTag(tag) is false, throw a RangeError exception.
  2. Let canonicalizedTag be CanonicalizeLanguageTag(tag).
  3. If NewTarget is undefined, let newTarget be the active function object, else let newTarget be NewTarget.
  4. Let locale be OrdinaryCreateFromConstructor(newTarget, %LocalePrototype%).
  5. Return InitializeLocale(locale, canonicalizedTag, options).

1.4Properties of the Intl.Locale Prototype Object#

The Intl.Locale prototype object is itself an Intl.Locale instance as specified in 1.5, whose internal slots are set as if it had been constructed by the expression Construct(%Locale%).

In the following descriptions of functions that are properties or [[Get]] attributes of properties of %LocalePrototype%, the phrase "this ListFormat object" refers to the object that is the this value for the invocation of the function; a TypeError exception is thrown if the this value is not an object or an object that does not have an [[initializedLocale]] internal slot with value true.

1.4.1Intl.Locale.prototype.constructor#

The initial value of Intl.Locale.prototype.constructor is %Locale%.

1.4.2Intl.Locale.prototype[ @@toStringTag ]#

This property has the attributes { [[Writable]]: false, [[Enumerable]]: false, [[Configurable]]: true }.

1.4.3Intl.ListFormat.prototype.format ([ list ])#

Intl.ListFormat.prototype.format is an accessor property whose set accessor function is undefined. Its get accessor function performs the following steps:

  1. Let lf be this value.
  2. If Type(lf) is not Object, throw a TypeError exception.
  3. If lf does not have an [[InitializedListFormat]] internal slot, throw a TypeError exception.
  4. If list is not provided or is undefined, then
  5. Return an empty String.
  6. Let x be ? ToArray(list).
  7. Return ? FormatList(lf, x).
