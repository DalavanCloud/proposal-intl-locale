DeconstructLanguageTag (_tag_)

The abstract operation DeconstructLanguageTag an argument _tag_. It returns a Record with a resolved locale and an extension.

1. Let _result_ be a new Record. 1. Let _result_.[[Locale]] be the String value that is _locale_ with all Unicode locale extension sequences removed. 1. Let _result_.[[Extension]] be the String value consisting of the first substring of _tag_ that is a Unicode locale extension sequence. 1. Return _result_.

IsOptionValueSupported (_option_, _value_)

The abstract operation IsOptionValueSupported accepts the arguments _option_ and _value_. It returns if the given value is a supported value for the given option.

1. If _option_ equals `"calendar"`, then: 1. If, for any locale _locale_, %DateTimeFormat%.[[LocaleData]].[[<_locale_>]].[[ca]] contains _value_, then, 1. Return *true*. 1. Else, return *false*. 1. If _option_ equals `"collation"`, then: 1. If, for any locale _locale_, %Collator%.[[LocaleData]].[[<_locale_>]].[[co]] contains _value_, then, 1. Return *true*. 1. Else, return *false*. 1. If _option_ equals `"hourCycle"`, then: 1. If _value_ is one of « `"h11"`, `"h12"`, `"h23"`, `"h24"` », then, 1. Return *true*. 1. Else, return *false*. 1. If _option_ equals `"caseFirst"`, then: 1. Assert: %Collator%.[[RelevantExtensionKeys]] contains `"kf"`. 1. If _value_ is one of `"upper"`, `"lower"`, `"false"`, 1. Return *true*. 1. Else, return *false*. 1. If _option_ equals `"numeric"`, then: 1. Assert: %Collator%.[[RelevantExtensionKeys]] contains `"kn"`. 1. If _value_ is *true* or *false*, then, 1. Return *true*. 1. Else, return *false*. 1. If _option_ equals `"numberingSystem"`, then: 1. If, for any locale _locale_, %NumberFormat%.[[LocaleData]].[[<_locale_>]].[[nu]] contains _value_, then, 1. Return *true*. 1. Else, return *false*. 1. Assert: Unreachable.

The Intl.Locale Constructor

The Locale constructor is a standard built-in property of the Intl object. Behaviour common to all service constructor properties of the Intl object is specified in .

Intl.Locale(tag[, options])

When the *Intl.Locale* function is called with an argument _tag_ and an optional argument _options_, the following steps are taken:

1. If NewTarget is *undefined*, throw a *TypeError* exception. 1. Let _locale_ be ? OrdinaryCreateFromConstructor(NewTarget, *%LocalePrototype%*). 1. If IsStructurallyValidLanguageTag(_tag_) is *false*, throw a *RangeError* exception. 1. Let _canonicalizedTag_ be CanonicalizeLanguageTag(_ctag_). 1. Let _r_ be DeconstructLanguageTag(_canonicalizedTag_). 1. Perform ? CreateDataPropertyOrThrow(_locale_, `"locale"`, _r_.[[Locale]]) 1. If _options_ is *undefined*, then 1. Let _options_ be ObjectCreate(*null*). 1. Else 1. Let _options_ be ? ToObject(_options_). 1. For each _key_ from the Option column of , do 1. If _key_ is `"caseFirst"` and %Collator%.[[RelevantExtensionKeys]] contains `"kf"`, or if _key_ is `"numeric"` and %Collator%.[[RelevantExtensionKeys]] contains `"kn"`, or if _key_ is neither `"caseFirst"` nor `"numeric"`, 1. Let _value_ be ? Get(_options_, _key_). 1. If _value_ is not *undefined*, 1. If the result of IsOptionValueSupported(_key_, _value_) is *false*, throw a *RangeError* exception. 1. Perform ? CreateDataPropertyOrThrow(_locale_, _key_, _value_). 1. If _r_.[[Extension]] is not *undefined*, then 1. Let _unicodeSubtags_ be CreateArrayFromList(UnicodeExtensionSubtags(_r_.[[Extension]])). 1. Let _i_ be *0*. 1. Let _len_ be ? ToLength(? Get(_unicodeSubtags_, `"length"`)). 1. Repeat while _i_ < _len_: 1. Let _key_ be Get(_unicodeSubtags_, ToString(_i_)). 1. If _key_ matches a Extension key column of the row in , then, 1. Increment _i_ by 1. 1. Let _value_ be ? Get(_unicodeSubtags_, ToString(_i_)). 1. If _value_ equals `"true"`, then 1. Set _value_ to *true*. 1. If _value_ equals `"false"`, then 1. Set _value_ to *false*. 1. Let _name_ be the value from a column Option in , which corresponds to the value _key_ from column Extension keys. 1. If the result of IsOptionValueSupported(_name_, _value_) is *false*, throw a *RangeError* exception. 1. If ? HasOwnProperty(_locale_, _name_) is *false*, then 1. Perform ? CreateDataPropertyOrThrow(_locale_, _name_, _value_). 1. Increment _i_ by 1. 1. SetIntegrityLevel(_locale_, `"frozen"`).
